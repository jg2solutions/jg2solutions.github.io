Somos muito transparentes em todo o processo de desenvolvimento do software e utilizamos muitas tecnologias pra auxiliar na comunicação e no gerenciamento do ciclo de vida. Confira como podemos te ajudar:

  • Video conferências periódicas via Hangouts ou Skype
  • Comunicação instantânea do projeto via Slack
  • Versionamento do código com o GitLab
  • Deploy automatizado com o Travis.CI
  • Gerenciamento de atividades com Trello

Nós estamos sempre investindo em capacitação profissional para melhor realizar nossos projetos. Abaixo está uma lista das tecnologias que utilizamos:

  • Linguagens: HTML, CSS, Javscript,PHP, C#, Java, Arduino
  • Frameworks: Angular, Ionic, Node.js, Jasmine, PHP Unit
  • Integração contínua: Travis, Jenkins, GitLab, GitHub.
  • Cloud Computing: Amazon AWS e Digital Ocean
  • Armazenamento de dados: MySQL, SQL Server, MongoDB, Firebase, Neo4j
